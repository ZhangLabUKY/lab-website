Cytokine data science

Current tools support biologically informed quality control, data transformation decisions, exploratory analysis, and multivariate approaches for cytokine datasets.

02

Continuous glucose monitoring

Current projects address CGM data quality, missing-data imputation, clinically relevant metrics, visualization, and reproducible exports.

Accessible statistical tools

R packages and Shiny applications translate complex workflows into forms that can be used by a broader research community.
